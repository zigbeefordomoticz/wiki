# The Web Interface — General Presentation

The plugin includes a built-in web interface accessible from your browser. It provides a graphical way to manage devices, monitor the Zigbee network, and configure the plugin without editing configuration files directly.

## Accessing the Web Interface

Please refer to [Step 3 — Plugin Configuration](Plugin_Configuration.md) for instructions on how to enable and access the web interface.

Once configured, the interface is available at:

```
http://<domoticz-ip>:<port>/z4d
```

## Sections

The web interface is divided into the following sections:

| Section | Purpose |
|---------|---------|
| [Dashboard](WebUI_Dashboard.md) | Overview of connected devices and their current status |
| [Management](WebUI_Management.md) | Add, remove, and configure devices and groups |
| [Network](WebUI_Network.md) | Zigbee network topology map and channel information |
| [Admin](WebUI_Admin.md) | OTA firmware updates, coordinator management, provisioning |
| [Settings](WebUI_Settings.md) | Plugin configuration parameters |
| [Tools](WebUI_Tools.md) | Diagnostic tools, raw Zigbee commands, log viewer |
| [About](WebUI_About.md) | Plugin version and system information |
| [Statistics](WebUI_Statistics.md) | Transport and network statistics |

*The interface appearance may have changed since this documentation was written.*
